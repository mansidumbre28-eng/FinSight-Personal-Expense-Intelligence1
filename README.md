# FinSight – Personal Expense Intelligence

AI-powered expense tracking and analytics dashboard using Python, Streamlit, and Machine Learning.

## Features

- Expense Tracking
- Income Management
- AI Expense Prediction
- Budget Planning
- Interactive Dashboard
- Smart Insights
- Analytics & Reports
- Savings Goals

## Technologies

- Python
- Streamlit
- Pandas
- NumPy
- Plotly
- Scikit-learn

## Run Project

```bash
pip install -r requirements.txt
streamlit run app.py
```
